# List Component
---

This component accepts children and wraps each child with a `<List.Item>` base component.

## Getting Started

```
import { List } from 'Templates/components';

<List>
 ...
</List>
```

## Props

### children (required)

_Type_: `node`  

###### Usage:

```
<List>
  ...
</List>
```
